
安装环境：
1、安装node.js：官网下载
2、安装ws（github开源nodejs websocket：https://github.com/websockets/ws）：sudo npm install ws
3、创建node.js-server：server.js
4、创建client：client.html（负责在浏览器发送消息，terminal接收消息）

执行：
1、terminal进入server.js目录，运行：node server.js
2、浏览器运行：client.html
---
3、停止服务：control+c
